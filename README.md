# AcadAI

AI-powered academic advisor system for students.

## Features
- GPA/GWA calculator
- AI subject recommendations
- Thesis suggestions
- Career matching

## Tech Stack
- HTML/CSS/JS
- Node.js backend
- AI API integration

## How to run
1. npm install
2. node server.js
3. open localhost:3000

IMPORTANT: Do NOT open index.html by double-clicking.
Always use http://localhost:3000
